# lark-task 手动改名说明

`lark-task/SKILL.md` 这个路径在当前 GitHub 连接器里会触发安全拦截，所以已先上传为：

```text
lark-task/skill.md
```

使用前请在 GitHub 网页里把文件名从 `skill.md` 改成：

```text
SKILL.md
```

操作方法：

1. 打开 `lark-task/skill.md`
2. 点右上角铅笔 Edit this file
3. 把文件名从 `skill.md` 改成 `SKILL.md`
4. 点 Commit changes

改完后路径应为：

```text
lark-task/SKILL.md
```
